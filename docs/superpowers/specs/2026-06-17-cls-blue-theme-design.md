# CLS Topic 切换 — 蓝色调浅色主题

**日期：** 2026-06-17  
**状态：** 已实现（v1.2.0）  
**文件：** `cls.js`  
**关联 spec：** [CLS UI 玻璃拟态](./2026-06-17-cls-ui-glass-design.md)

## 背景

v1.1.x 悬浮区与菜单均为白色，在 CLS 控制台页面上显得突兀，且与背后内容对比不佳。用户要求悬浮区 + 菜单整体不再使用白色，采用与 CLS 蓝色系统一的浅色主题。

## 目标

1. 悬浮触发区 + 菜单统一蓝灰色调，无白色大块
2. 全部实色背景，避免透明透出页面内容
3. 主按钮实心腾讯蓝，与 rancher 操作习惯一致
4. 仅改 `cls.js`，不动 `rancher.js`

## 色板

| 令牌 | 色值 | 用途 |
|------|------|------|
| `--cls-surface` | `#f4f8ff` | 菜单/列表/贴边标签底 |
| `--cls-surface-raised` | `#eaf1ff` | 分区标题、底栏、工具条小钮 |
| `--cls-surface-hover` | `#e8f0fe` | 列表悬停 |
| `--cls-surface-active` | `#dbeafe` | 当前 Topic 高亮 |
| `--cls-accent` | `#006EFF` | 主按钮、强调色 |
| `--cls-border` | `#c8daf0` | 边框 |
| `--cls-text` | `#1a2a4a` | 主文字 |
| `--cls-text-muted` | `#5a6a8a` | 次要文字 |
| `--cls-input-bg` | `#ffffff` | 表单输入框（唯一白底） |

## 组件

| 组件 | 样式 |
|------|------|
| 主按钮 | 实心 `#006EFF`，白图标，蓝色阴影 |
| 工具条小钮 | `#eaf1ff` 底，深蓝灰图标 |
| 贴边标签 | `#f4f8ff` 底，蓝字蓝边 |
| 菜单容器/列表/列表项 | `#f4f8ff` |
| 分区/底栏 | `#eaf1ff` |
| 当前 Topic | `#dbeafe` + 左 3px 蓝条 |
| 表单输入 | 白底保证可读 |
